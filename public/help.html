Welcome to Simple Chat

Simple Chat is a simplistic chatting application, with a few additional features which may be helpful.

You can join any room, or can make your own.

This is the list of available commands:

  • /cc : Clear chat
  • /typing : Toggle typing notifications
  • /ping user : Ping another user
  • /pingServer : Ping the server (under construction)
  • /help : See this list.

There is also a profanity filter in all rooms
